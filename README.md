# 📌 Multer Task Manager

A Node.js task manager with file upload using Multer, JWT authentication, and CRUD operations.

---

## 🚀 Features

- 🔐 User Signup & Login (JWT Authentication)
- 📂 File Upload (Images / PDFs) using Multer
- 📝 Add Tasks with file attachments
- 🗑️ Delete Tasks
- 📄 View uploaded files
- 💾 Data stored using JSON (no database)

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Multer (file upload)
- JSON Web Tokens (JWT)
- bcrypt (password hashing)
- HTML, CSS (Frontend)

## ⚙️ How to Run

1. Install dependencies:

```bash

```

2. Start Server:

```bash
node server/app.js
```

3. Open in browser:

Open `client/login.html` in your browser
